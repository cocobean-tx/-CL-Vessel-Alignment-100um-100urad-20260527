EUV光源多层钼硅膜收集镜与腔体PF点对准装调方案

基于PSM、定心球面镜、PF靶球及平行双V槽+销钉定位的可见光装调工艺

文档密级：内部 · 版本：V3.0 · 精度版本③：位置 ≤ ±0.1 mm，姿态 ≤ ±0.1 mrad · 日期：2026-05-26

1. 方案总述与目标

本方案的最终对准目标是使收集镜（CL）的光学PF点与光源腔体的真实PF点重合，并精确获取该PF点的空间坐标，为后续锡滴打靶实验提供关键输入。光源腔体在整个过程中完全固定，收集镜通过两个平行V槽加一个销钉实现高精度定位安装。

核心装调逻辑：①将定心球面镜通过转接工装安装于收集镜接口（平行双V槽+销钉），利用PSM共焦标定腔体IF点，锁定PSM后拆除球面镜及工装；②收集镜以相同方式安装于同一V槽接口，其PF点与腔体设计PF点基本重合（偏差由V槽定位精度及离线标定保证）；③将PF靶球移入光路，基于IF→PF共轭关系，调节靶球位置使自准直光斑达到预设的参考直径D_ref并居中，此时靶球球心与收集镜PF点精确重合，即标记了腔体真实PF点，锁定靶球并记录空间坐标；④锁紧收集镜并复测。整个过程以光斑图像为唯一判据，不依赖位移台读数或设计值计算。

2. PF点定位判据的重要说明

已知PF靶球直径约10~15 mm，曲率半径R_ball≈5~7.5 mm，而PF-IF之间的机械设计距离为1886.927 mm。此距离远大于靶球曲率半径（约三个数量级）。在此条件下，靶球表面反射的光斑形态受到球面曲率的强烈影响，近轴近似公式 D≈2·NA·d 不再成立。当靶球球心沿轴向移动时，返回光斑直径的变化并非简单的线性关系，且最小光斑位置并不对应靶球球心在PF点。因此，不能通过寻找最小光斑来定位PF点。必须通过离线灵敏度标定，实测该特定靶球在设计距离d_design下的光斑参考直径D_ref，作为阶段二的定量判据。

3. 关键器件定义与功能

器件名称核心功能安装约束与工作条件

激光器为PSM提供照明光源。波长635 nm，功率≥10 mW（光纤耦合输出），功率稳定性≤0.5% RMS，光谱宽度≤1 nm，单模，FC/PC或SMA905接口。

平行双V槽+销钉机构用于精确定位CL及转接工装。V槽平面度≤0.001 mm，平行度≤0.0015 mm，销钉孔配合H6/g5，重复定位精度≤±2 μm。

4. 激光器波长选择

为什么优选635 nm：

  1. 人眼视觉灵敏度：人眼光谱光视效率函数（明视觉）峰值约为555 nm（绿色）。在红光波段，635 nm的人眼相对灵敏度约为650 nm的2~3倍，约为670 nm的5倍以上。因此在同等辐射功率下，635 nm激光比650/660 nm激光在人眼中亮度高约2~3倍，更易于装调过程中肉眼观察和判断光斑形态。在PSM装调过程中，操作人员需要直接观察光斑进行初步对准，635 nm提供更好的可见性。
  2. 可见光对准兼容性：EUV工作波长为13.5 nm，无法直接用于可见光装调。可见光装调必须在EUV波段之外的可见光波段进行。635 nm处于可见光红-橙区域（人眼敏感区），是精密光学装调领域广泛采用的基准波长之一。
  3. 膜系匹配性：定心球面镜和PF靶球均需镀制高反射膜。635 nm波段的介质高反膜工艺成熟，反射率可达90%以上（如TiO₂/SiO₂或Ta₂O₅/SiO₂多层膜系），成本低、可靠性高。
  4. 激光器可获得性：635 nm半导体激光器技术成熟，单模光纤耦合输出功率≥10 mW的模块供应充足，功率稳定性（≤0.5% RMS）和光谱宽度（FWHM≤1 nm）指标均可在货架产品中获得。
  5. 其他可见光波长的可行性分析：
    候选波长优势劣势是否可选
    532 nm（绿光）人眼最敏感，亮度最高散斑干扰严重，影响质心提取，膜系需单独设计可选但不推荐
    650/660 nm成本低人眼灵敏度低，装调效率差非优选
    405 nm短波长，光斑小不可见，安全隐患不可选
    780/850 nm成本低不可见不可选
  6. 结论：综合人眼灵敏度、膜系成熟度、器件可获得性及装调操作便利性，优选635 nm。其他可见光波长在特定条件下可用，但均存在显著劣势。532 nm的散斑问题在亚微米级定位中尤为突出——散斑对比度高会导致质心提取抖动增大（可达0.2~0.5像素），直接降低定位精度，而635 nm半导体激光器的相干长度较短，散斑对比度显著低于DPSS绿光激光器。因此，635 nm是最优选择。

5. 总体装调阶段划分

  • 阶段一：标定光源腔体IF点。定心球面镜转接工装安装于V槽接口，PSM通过调节机构共焦标定真实IF点并锁定，拆除工装。
  • 阶段二：收集镜安装与靶球定位腔体PF点。CL通过同一V槽接口安装就位，移入PF靶球，通过三维位移台调节靶球使自准直光斑达到D_ref且居中，锁定靶球并记录坐标。
  • 阶段三：锁紧复测与精度验证。锁紧收集镜，复测光斑稳定性，验证对准精度。

6. 定心球面镜球心与IF点重合的机械保证

定心球面镜通过转接工装安装于收集镜接口（平行双V槽+销钉），其球心与腔体理论IF点的重合由以下机械设计保证：

  • 接口基准统一：转接工装的底部定位面（V槽配合面和销钉孔）与收集镜完全相同，利用同一套V槽和销钉定位，确保重复拆装后空间位置一致。
  • 工装精度设计：工装采用因瓦合金制造，关键尺寸公差控制在±0.002 mm以内。球面镜安装座的位置相对于工装底部基准面经精密标定，其理论位置由IF点坐标和接口基准面坐标计算得出。
  • 首次校准与锁死：工装上设有微调机构（差分顶丝或楔形垫片），首次安装时利用PSM共焦信号进行在线校准，使球心精确位于IF点，然后锁死微调机构并注入固定胶。此后工装成为"IF点参考件"，重复拆装不再需要重新校准。
  • 热稳定性：因瓦合金的热膨胀系数极低（≤1.5×10⁻⁶/K），在装调环境温度（22±0.1℃）下，工装热变形引起的球心漂移可忽略不计。

通过以上措施，定心球面镜球心与腔体IF点的最终偏差可控制在≤±10 μm以内，满足IF点标定误差预算要求。

7. CL与Vessel六自由度偏差计算方法

CL（收集镜）相对于vessel（光源腔体）在X/Y/Z、Rx/Ry/Rz六个自由度的偏差，通过以下方法计算：

7.1 坐标系统定义

  • 以腔体V槽和销钉接口为基准，建立腔体坐标系 {V}：原点位于销钉孔中心，X轴沿两V槽对称中心线，Y轴垂直于V槽底面向上，Z轴按右手定则确定。
  • CL坐标系 {C}：以CL的PF点为原点，定位基准面（V槽配合面和销钉孔）为参考。

7.2 平动偏差（X/Y/Z）计算

方法一——离线标定法（主方法）：CL在其独立坐标系{C}中的PF点坐标通过离线干涉测量或坐标测量机（CMM）精密标定，精度≤±1 μm。CL安装于V槽后，CL坐标系{C}相对于腔体坐标系{V}的理论变换矩阵 T_C^V（6自由度）由机械设计给出。实际安装后，PF点在{V}中的坐标为：

P_PF^V = T_C^V · P_PF^C

平动偏差 ΔX, ΔY, ΔZ 为实际测量值与设计值的差值。其中ΔX, ΔY通过阶段二靶球定位时的光斑质心偏移量反算；ΔZ通过D_ref标定曲线上的光斑直径偏离量反算。

方法二——CMM直接测量法（验证法）：在CL安装于V槽后，使用CMM直接测量CL外缘基准面或标记点相对于腔体基准的位置，通过坐标变换解算六自由度偏差。精度取决于CMM精度（典型≤±0.5 μm + L/500 mm）。

7.3 角度偏差（Rx/Ry/Rz）计算

  • Rx, Ry（俯仰/偏摆）：
    • 方法A——光斑质心偏移法：PSM的聚焦点位于IF点。当CL安装后，其PF点位置变化会改变返回光路。靶球置于理论PF点时，若Rx或Ry存在偏差，靶球球心将在横向上偏离光轴，导致返回光斑质心偏移。在小角度近似下，横向偏移量 Δx ≈ f_CL · tan(Ry) ≈ f_CL · Ry（Ry为小量）。由光斑质心偏移量和CL焦距f_CL，可解算Rx和Ry。
    • 方法B——自准直仪法：在CL背面或侧面粘贴小平面反射镜（基准面已知），使用电子自准直仪测量其法线方向，与腔体基准方向比较，直接获得Rx、Ry偏差。精度可达0.01″。
  • Rz（绕Z轴旋转）：
    • 方法C——销钉定位间隙法：Rz由销钉与孔的配合间隙决定。采用H6/g5配合，最大间隙更小。例如孔径φ6 H6(+0.008/0)，销钉φ6 g5(-0.004/-0.009)，最大间隙=0.008-(-0.009)=0.017 mm。L_pin≈100 mm时，Rz_max≈0.17 mrad。实测Rz可通过高精度CMM或激光跟踪仪测量CL侧面基准点获得。

7.4 综合偏差求解流程

  1. CL离线标定：记录CL坐标系{C}中PF点坐标及定位基准面参数。
  2. CL安装于V槽。
  3. 利用PSM+靶球系统（阶段二流程），获取靶球位于PF点时的光斑质心(X, Y)和直径D。
  4. 由光斑质心偏移反算ΔX, ΔY；由D与D_ref的偏差反算ΔZ。
  5. 由ΔX, ΔY结合CL焦距f_CL解算Rx, Ry（或采用自准直仪直接测量）。
  6. 通过CMM或销钉间隙分析法确定Rz。
  7. 输出六自由度偏差向量 [ΔX, ΔY, ΔZ, Rx, Ry, Rz]，作为锡滴打靶补偿输入。
注意：本偏差计算方法已整合到阶段三"锁紧复测与精度验证"流程中，作为对准精度验证的核心环节。六自由度偏差数据同时为后续锡滴打靶实验的空间坐标补偿提供关键输入。

8. 详细操作SOP

8.1 阶段一：标定光源腔体IF点

  1. 安装定心球面镜转接工装：工装底部具有与CL完全相同的平行双V槽配合面和销钉孔，以相同方式落座于腔体V槽上，销钉插入孔中定位，按规定力矩锁紧。
  2. 架设PSM及调节机构：将PSM安装于三维平移+角位移组合台上，再将组合台固定于腔体外侧稳定基座。初步调节平移和角度，使PSM大致对准球面镜。
  3. 粗调PSM位置与姿态：打开激光器，通过相机观察反射光斑。调节PSM的三维平移和角度，使光斑进入相机视场并大致居中。
  4. 精调至共焦状态：精细调节PSM的轴向位置（调焦），使光斑达到最小（艾里斑形态），此时PSM聚焦点与球面镜球心（即IF点）精确共轭。确认光斑质心位于视场中心。
  5. 锁定PSM并记录：锁紧PSM调节机构的所有自由度，再次确认光斑状态不变。记录相机图像作为基准。此时PSM位置即为腔体真实IF点的光学基准。
  6. 拆除定心球面镜转接工装：小心拆下转接工装，不触碰PSM及调节机构。

8.2 阶段二：收集镜安装与靶球定位腔体PF点

  1. 安装收集镜：将CL以相同方式安装于同一V槽接口，销钉定位，按规定力矩锁紧。
  2. 安装PF靶球及位移台：将靶球通过安装杆固定于三维位移台上，位移台安装于腔体外部独立支架。初步调节靶球位置至理论PF点附近。
  3. 寻找初始参考点：将靶球移动至PSM聚焦点（IF点）附近，使光斑达到极小状态，记录位移台读数Z0。
  4. 扫描测量：以Z0为起点，将靶球向远离PSM方向逐点移动，覆盖设计d_design前后的范围（d_design=1886.927 mm，扫描范围1886~1890 mm），步长5 μm。在每个位置记录位移台读数Δz = Zi – Z0，并同步采集相机中的光斑图像。
  5. 提取光斑直径：对每帧图像使用高斯拟合算法提取光斑的半高全宽（FWHM）直径D。
  6. 绘制D-Δz曲线：以Δz为横坐标，D为纵坐标，拟合多项式曲线D = f(Δz)。
  7. 确定D_ref：在曲线上查出Δz = d_design对应的光斑直径D_ref。
  8. 验证灵敏度：在Δz = d_design处求导，得到实际灵敏度S_meas = dD/d(Δz)。

8.3 阶段三：锁紧复测与精度验证

  • 基准记录：阶段二锁定靶球时，记录光斑质心(X₀, Y₀)、直径D₀和靶球位移台坐标。
  • 锁紧偏差：锁紧CL后复测光斑质心，计算偏移Δr，换算物方PF点偏差δ = Δr/M。
  • 六自由度偏差计算：按照第7节所述方法，计算CL相对于vessel的六自由度偏差[ΔX, ΔY, ΔZ, Rx, Ry, Rz]。
  • 长期稳定性：24小时内每1小时记录一次光斑质心，漂移量应≤0.3 μm。
  • 坐标重复性：解锁靶球位移台并重新调节至D_ref共焦状态，重复5次，坐标偏差≤0.3 μm。

9. 参数确定逻辑

步骤参数确定方法
①IF点空间坐标由腔体光学设计确定，或通过机械测量标定。
②定心球面镜曲率半径R球面镜球心必须与IF点重合。球面镜表面位置由接口基准面+转接工装厚度决定。因此R = IF点到球面镜表面的轴向距离，由机械几何直接计算得出。
③PSM所需工作距离WD_req设定球面镜表面到PSM物镜前端的安全间隙d_safe（≥20 mm）。WD_req = R + d_safe。此值作为PSM选型的硬约束。
④PSM物方NA根据靶球轴向定位灵敏度需求（δz）、相机像素尺寸和放大率M，通过S_min = ΔD_obj/δz及NA_min≈S_min/2计算得出。
⑤定心球面镜口径Ds由R和NA决定：D_beam = 2×R×NA，Ds = 1.1×D_beam。
⑥PSM焦距ff≈M×WD_req。
⑦PSM调节机构指标行程覆盖机械偏差+调节余量；分辨率满足共焦精细调节需求；锁紧稳定性满足长期不漂移要求。
⑧靶球位移台指标行程覆盖安装偏差+调节范围；分辨率≤灵敏度/10；锁紧漂移远小于定位误差预算。

10. 对准精度测量方案

  • 基准记录：阶段二锁定靶球时，记录光斑质心(X₀,Y₀)、直径D₀和靶球位移台坐标。
  • 锁紧偏差：锁紧CL后复测光斑质心，计算偏移Δr，换算物方PF点偏差δ = Δr/M。
  • 六自由度偏差：按第7节方法计算CL相对于vessel的[ΔX, ΔY, ΔZ, Rx, Ry, Rz]。
  • 长期稳定性：24小时内每1小时记录一次光斑质心，漂移量应≤0.3 μm。
  • 坐标重复性：解锁靶球位移台并重新调节至D_ref共焦状态，重复5次，坐标偏差≤0.3 μm。
